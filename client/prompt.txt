Привет! В файле admin.html должна быть панель управления. Она включает в себя:

-- 1. Две кнопки Остановить очередь и Запустить очередь: они запускают ручки POST /donations/cancel-processing и POST /queue/next

-- 2. Последние 10 записей в очереди, можно получить из GET /donations/last10 и query status = PENDING, а также query status = IN_PROGRESS, и query status = COMPLETED,
Пусть это будут три таблички в ширину. Ответ GET метода выглядит как массив объектов:
{
    "id": "bed566d1-8449-4f9f-9f22-e79ef28be20d",
    "username": "SuperUser",
    "eventId": "718ddbb5-9498-4e48-a60d-514a93167cb4",
    "amount": 1000,
    "currency": "rub",
    "type": "rub",
    "status": "COMPLETED",
    "createdAt": "2025-05-20T15:52:40.000Z",
    "updatedAt": "2025-05-20T15:52:40.000Z",
    "message": ""
  },
  нам надо отображать первые 8 символов id, полный username и полностью amount + currency

  -- 3. Должен быть редактор конфигурации донатов
  Чтение: GET /dynamic-data/donations-config
  Запись: POST /dynamic-data/donations-config

  Пусть это будет таблица с редактируемыми полями

  {
    // От какой суммы
  from: number

  // До какой суммы
  to: number

  // Что выполнить в Terminal
  execute: string

  // Валюта: rub | usd
  currency: string

  // Включено или выключено
  enabled: boolean
}

Также должна быть кнопка сохранить в низу таблицы. Ну и кнопка обновить какая-нибудь, чтобы она заново считала данные в таблицу

-- 4. Должна быть кнопка "Отправить пожертвование вручную", она активирует POST /donations/create
Формат данных такой:
{
  "username": "SuperUser",
  "amount": 1000,
  "currency": "rub",
  "message": ""
}



Давай обсудим пару правил. Когда мы редактируем правила донатов from и to - это числа: целые или с плавающей точкой.

Оформление: Сделай красивый темный дизайн, заказчик у нас Даня Крастер для его твич канала, можешь подумать что-то в его цветовой гамме. Только не упоминай его на прямую.
И да, везде, вместо слова "донаты" используй "пожертвования". Говоря о теме продукта: Это что-то типо "программа, которая запускает скрипт в зависимости от суммы доната"
